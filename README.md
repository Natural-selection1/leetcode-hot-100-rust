## 📖 项目简介

本仓库用于记录自己使用 rust 编程语言书写的 [LeetCode 热题 Hot 100](https://leetcode.cn/studyplan/top-100-liked/) 的解决方案

每一道题目的解决方案都力求：

- **代码清晰**：易于理解和学习。
- **符合 Rust 特色**：充分利用 Rust 的语言特性, 并尽可能写出 Rust 风格的代码。

## 🚀 如何使用本仓库

1.  克隆仓库
2.  安装 VS Code LeetCode 插件并登录
3.  点击提交即可使用 leetcode OJ 判决相关代码
