## 项目简介

本仓库用于记录 [LeetCode 热题 Hot 100](https://leetcode.cn/studyplan/top-100-liked/) 的 rust 解决方案

每一道题目都力求：

- **代码清晰**：自解释性强、易于理解和学习
- **符合 Rust 风格**：充分利用 Rust 的语言特性、而不是其他语言的直接翻译

## 如何使用本仓库

1.  克隆仓库
2.  安装 VS Code LeetCode 插件、并使用 `LeetCode: Sign In` 登录
3.  点击文件尾部的 `Submit` 或使用 `LeetCode: Submit to LeetCode` 即可判决相关代码
